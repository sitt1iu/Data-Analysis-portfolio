本分析基于某电商平台 2014 年“双十二”大促前后用户行为数据，旨在对比活动期间与日常期间用户在平台上的行为差异。

重点关注以下指标与行为：

- PV（Page View）与 UV（Unique Visitors）的变化趋势

- 用户行为路径（浏览、收藏、加购、购买）

- 转化率与转化漏斗

通过对不同时间段用户行为的系统分析，期望为平台流量分发优化及商家营销策略提供数据支持。

除此之外，本分析还简略地观察了平台的留存率，以此分析平台的用户粘度、市场渗透率。

### 问题

双十二活动期间与日常期间，用户在平台上的行为模式有何差异？

### 数据

本分析使用的数据来源于阿里天池竞赛平台：  
https://tianchi.aliyun.com/competition/entrance/231522/introduction  

数据为用户在商品全集上的移动端行为记录（Dataset D），包含以下字段：

| 字段名 | 含义 | 说明 |
|--------|------|------|
| user_id | 用户标识 | 抽样 & 脱敏处理 |
| item_id | 商品标识 | 脱敏处理 |
| behavior_type | 用户行为类型 | 1=浏览，2=收藏，3=加购物车，4=购买 |
| user_geohash | 用户地理位置 | 由经纬度加密生成，可为空 |
| item_category | 商品分类 | 脱敏处理 |
| time | 行为时间 | 精确到小时级 |


## 建议

1. 虽然默认用户的购买路径是‘点击 - 加购 - 收藏 - 购买’，实际上我们在不同时期用户行为分析中看到的，在大促期间，购买的数量要明显超过收藏的数量。平台需要考虑‘收藏’这个功能在整个路径中的作用，是否是用户强烈购买意愿的表示。如果不是的话，是否可以优化。

2. 根据日均各时间段的用户行为变化，我们可以得出，在活动期间，用户在晚上九点到零点之间都很活跃，且因为商家大促多集中在零点，购买行为在零点达到最高值，因此建议商家在晚上八点之前将商品折扣页面放出，保证足够的曝光率；而非活动期间，用户在晚上九点至十点之间最活跃，商家要抓住这个时间段宣传或者搞活动。

3. 从用户的转化漏斗来看，从点击到加购的转化率最低，在这个阶段流失了大部分的用户。因此想要下一步提升购买率，商家要优化从点击到加购这一转化。

4. 从留存率来看，大促期间的客源并非只是来薅羊毛的一次性用户，用户平时也在用这个平台进行购物，只是大促将他们集中在这段时间进行消费。总的来说，该电商品台在市场上的渗透率还是很可观的。
